A Temperatura Recente de um Projeto de Lei

Conceito de Temperatura

A temperatura de uma proposio um conceito criado com o objetivo de medir quo movimentada est a tramitao de um Projeto de Lei num determinado perodo, indicando quanto esforo tem sido empregado por atores do processo legislativo em avanar/barrar o projeto.

Clculo da temperatura

A temperatura de um PL pode ser calculada com diferentes granularidades/perodos (dia, semana, ms). Alm disso, possvel escolher o tamanho da janela do histrico recente a ser considerado no clculo (ex: os ltimos 22 dias, ou as ltimas 4 semanas). A temperatura de um perodo definida como a quantidade de eventos ocorridos nesse perodo ponderados, estes pesos so dados de acordo com o tipo de evento (despacho, discusso e votao) alm do local do evento tambm influenciar o peso (plenrio, ccj, cft, comisses especiais e cae tm um peso maior). A funo de temperatura dada pela frmula:

\[t = \sum_{j=0}^{n} (v_j)\alpha^j\]

onde:

  • n o tamanho da janela mvel a ser utilizada
  • j o ndice da janela mvel a ser utilizada
  • \(\alpha\) a taxa de decaimento (deve estar entre 0 e 1);
  • \(v_j\) a quantidade de eventos ponderados (requerimentos, votaes, etc) do perodo de ndice \(j\) na janela;
  • \(t\) a temperatura recente do perodo;

Exemplo

Vamos calcular a temperatura para um Projeto de Lei de exemplo.

Escolhemos o PL 490/2007, que trata da demarcao de terras indgenas. Vamos baixar os dados da proposio e da tramitao e em seguida reconhecer os eventos e fases, mostrando os ltimos eventos ocorridos na tramitao do PL.

#devtools::install_github('analytics-ufcg/agora-digital') #Descomentar para instalar o pacote
library(magrittr)
id <- 345311
casa <- 'camara'
prop <- agoradigital::fetch_proposicao(id,casa)
tram <- agoradigital::fetch_tramitacao(id,casa)
proc_tram <- agoradigital::process_proposicao(prop,tram,casa)
proc_tram %>%
  dplyr::select(data_hora, texto_tramitacao, sigla_local, evento) %>%
  dplyr::filter(!is.na(evento)) %>%
  utils::tail(6) %>%
  DT::datatable(rownames = F, data = )

Temperatura Semanal

Vamos calcular a temperatura recente para todo o histrico da tramitao do PL, agregando os eventos por semana e utilizando um decaimento de 10%. Os resultados podem ser vistos no grfico abaixo.

Temperatura Semanal (pontos)

Os pontos pretos representam os valores da temperatura recente calculada para cada semana.

pautas <- tibble::frame_data(~data, ~sigla, ~id_ext,    ~local, ~casa,  ~semana,    ~ano)
hist_temperatura_semanal <- agoradigital::get_historico_temperatura_recente(proc_tram, granularidade = 's', decaimento = 0.1, pautas = pautas)

ggplot2::ggplot(hist_temperatura_semanal, ggplot2::aes(x=periodo, y=temperatura_recente)) +
  ggplot2::xlab("Tempo") + 
  ggplot2::ylab("Temperatura") +
  ggplot2::geom_point() +
  ggplot2::theme_minimal()

Temperatura Semanal (linha)

Para ter uma visualizao menos poluda, vamos mostrar uma linha, ao invs dos pontos.

hist_temperatura_semanal %>% 
  ggplot2::ggplot(ggplot2::aes(x=periodo, y=temperatura_recente)) +
  ggplot2::geom_line() +
  ggplot2::xlab("Tempo") + 
  ggplot2::ylab("Temperatura") +
  ggplot2::theme_minimal()

Temperatura Semanal (curva suavizada)

Agora vamos aplicar uma suavizao curva de temperatura.

hist_temperatura_semanal %>% 
  ggplot2::ggplot(ggplot2::aes(x=periodo, y=temperatura_recente)) +
  ggplot2::geom_smooth(span = .05, se = F) +
  ggplot2::xlab("Tempo") + 
  ggplot2::ylab("Temperatura") +
  ggplot2::theme_minimal()

Temperatura Semanal (curva suavizada - ltimos 3 meses)

Podemos tambm ter uma visualizao mais detalhada, olhando apenas os ltimos 3 meses (~ 12 semanas) da tramitao do PL, e mostrando a quantidade de eventos ocorridos a cada semana.

Os pontos vermelhos representam o nmero de eventos ocorridos em cada semana na qual houve eventos relevantes na tramitao do PL.

start_date <- dplyr::last(proc_tram$data_hora) - lubridate::days(90)
hist_temperatura_semanal_3_meses <- hist_temperatura_semanal %>% 
  dplyr::filter(periodo >= start_date)

hist_temperatura_semanal_3_meses %>% 
  ggplot2::ggplot(ggplot2::aes(x=periodo, y=temperatura_recente)) +
  ggplot2::geom_smooth(span = .3, se = F) +
  ggplot2::geom_point(ggplot2::aes(x=periodo, y=temperatura_periodo), colour='red', alpha=0.5, subset(hist_temperatura_semanal_3_meses, temperatura_periodo > 0)) +
  ggplot2::xlab("Tempo") + 
  ggplot2::ylab("Temperatura") +
  ggplot2::theme_minimal()
